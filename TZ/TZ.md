## 4) Функциональные требования (поведение)

### 4.1. Переключение темы (dark/light)

- Кнопка: плавающая `data-theme-toggle` (правый нижний угол).
- Тема задаётся атрибутом `data-theme` на `html`.
- Значение сохраняется в `localStorage` по ключу **`yb_theme`**.
- При первом заходе (если нет сохранения) выбирается по `prefers-color-scheme`.
- Учитывать `prefers-reduced-motion` (часть анимаций отключается).

### 4.2. Якорная навигация и плавный скролл

- Переходы по `#...` внутри страницы.
- Глобально включён `scroll-behavior: smooth;` (в CSS).

### 4.3. Анимации и эффекты

- **Scroll‑reveal**: элементы с `data-animate` получают класс `is-inview` через `IntersectionObserver`.
- **Tilt**: 3D‑наклон блока с `data-tilt` при движении мыши.
- **Кастомный курсор**:
  - включается на устройствах с hover (не touch)
  - скрывает системный курсор (`body { cursor: none; }`)
  - реагирует на элементы с классом `.cursor-target`
  - GSAP‑анимации: вращение/обводка углов, scale на mousedown/mouseup
- **Фоновые анимации**:
  - `#color-bends`: WebGL/Three.js shader‑фон
  - `#silk-bg`: canvas “silk strands” + виньетка
  - отключаются при `prefers-reduced-motion: reduce`

## 5) Дизайн и UI (текущее состояние)

- Общий стиль: **dark neon / cosmic**, с поддержкой **light theme** (через CSS‑переменные).
- Основные шрифты: **Manrope** и **Outfit** (Google Fonts).
- Основные акценты: бирюзовый / фиолетовый / розовый / тёплый оранжевый.
- Визуальные паттерны:
  - “glassmorphism” карточек (полупрозрачные поверхности, blur)
  - мягкие градиенты, glow, “noise” слой
  - hover‑подсветка карточек и лёгкое смещение

## 6) Адаптивность и доступность (как реализовано)

- Адаптивность:
  - бургер‑меню на ширине до ~860px
  - перестройка сеток (hero, reviews, works, projects, directions, contact) в 1 колонку на соответствующих брейкпоинтах
- Доступность (частично):
  - `aria-label` у навигации, burger, corner mark
  - у квиза радиогруппы помечены `role="radiogroup"` + `aria-label`
  - учитывается `prefers-reduced-motion`
  - кастомный курсор отключается на touch
